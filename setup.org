* Commands
** Dmenu Replacement
[[https://github.com/davatorium/rofi][rofi]] is a good =dmenu= allowing a lot of configuration. Its configuration file, =~/.config/rofi/rofi.rasi=, is generated in [[/home/leo/dotfiles/rofi.org][rofi.org]].

#+NAME: dmenu
#+BEGIN_SRC python
return "rofi -dmenu -config ~/.config/rofi/rofi.rasi -i -p"
#+END_SRC

#+NAME: drun
#+BEGIN_SRC python
return "rofi -show drun -config ~/.config/rofi/rofi.rasi"
#+END_SRC
